Healthy Recipes App

This project is an Android application built using Jetpack Compose as part of a University of Greenwich coursework.

 Features
- Displays a list of healthy recipes
- Recipe detail screen with ingredients and steps
- Light and dark mode toggle
- Navigation between screens using Jetpack Navigation

Technologies Used
- Kotlin
- Jetpack Compose
- Material 3
- Android Studio

How to Run
1. Open the project in Android Studio
2. Sync Gradle
3. Run on an emulator or physical device

Notes
This app was rebuilt and enhanced from a basic starter version as required by the coursework brief.
